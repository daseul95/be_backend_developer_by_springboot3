
IoC(제어의 역전)
다른 객체를 직접 생성, 제어 하는것이 아니라 외부에서 관리하는 객체를
가져와 사용하는 것을 말함.

DI(의존성 주입)
어떤 클래스가 다른 클래스에 의존한다는 뜻

빈?
스프링 컨테이너에서 관리하는 객체

기존의 자바 코드는 클래스A에서 B객체를 쓰고 싶은 경우 직접 생성
그러나 스프링의 경우 스프링 컨테이너에서 객체를 주입 받아 사용합니다. -->Ioc/DI 개념


스프링 컨테이너?
스프링이 제공함

다시 한번 빈이란?
스프링 컨테이너가 생성하고 관리하는 객체
빈을 등록하는 방법은 여러가지가 있습니다(XML파일 설정,에너테이션 추가 등)
어렵게 생각하지 않을려면 스프링의 객체라고 생각하면 편합니다.


스프링에서 중요한 어노테이션 3가지
@SpringBootConfiguration   //스프링 부트 관련 설정을 나타냄
@ComponentScan             //사용자가 등록한 빈을 읽고 등록함
                           //용도에 따라 다른 애너테이션도 사용하는 경우가 많음
@EnableAutoConfiguration   //스프링구성 자동구성을 활성화
